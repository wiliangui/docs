# Get account balance

`GET /api/v1/balance`

Retrieves your current account balance and available funds.

## Request

### Headers

| Header | Type | Required | Description |
|--------|------|----------|-------------|
| `X-API-Key` | string | Yes | Your API key |
| `X-API-Secret` | string | Yes | Your API secret |

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful.
</ResponseField>

<ResponseField name="data" type="object">
  The balance data.
  
  <Expandable title="data properties">
    <ResponseField name="available" type="number">
      Available balance that can be withdrawn.
    </ResponseField>
    
    <ResponseField name="pending" type="number">
      Pending balance from transactions being processed.
    </ResponseField>
    
    <ResponseField name="total" type="number">
      Total balance (available + pending).
    </ResponseField>
    
    <ResponseField name="currency" type="string">
      Currency code (always "BRL" for Brazilian Reais).
    </ResponseField>
    
    <ResponseField name="lastUpdated" type="string">
      ISO 8601 timestamp of last balance update.
    </ResponseField>
  </Expandable>
</ResponseField>

## Status codes

| Status | Description |
|--------|-------------|
| `200` | Balance retrieved successfully |
| `401` | Invalid or missing authentication |
| `500` | Internal server error |

## Understanding balance types

### Available balance

The amount you can withdraw immediately. This includes:
- Completed payments
- Minus any fees
- Minus pending withdrawals

### Pending balance

Funds that are being processed and not yet available for withdrawal. This includes:
- Payments in processing
- Funds under review
- Scheduled transfers

### Total balance

The sum of available and pending balances, representing your total account value.

## Example use cases

### Check before withdrawal

```javascript Node.js
async function requestWithdrawal(amount) {
  // Check balance first
  const balanceResponse = await fetch('https://api.primepix.me/v1/balance', {
    headers: {
      'X-API-Key': process.env.PRIMEPIX_API_KEY,
      'X-API-Secret': process.env.PRIMEPIX_API_SECRET
    }
  });
  
  const { data: balance } = await balanceResponse.json();
  
  if (balance.available < amount) {
    throw new Error(`Insufficient balance. Available: ${balance.available}`);
  }
  
  // Proceed with withdrawal
  const withdrawResponse = await fetch('https://api.primepix.me/v1/withdraw', {
    method: 'POST',
    headers: {
      'X-API-Key': process.env.PRIMEPIX_API_KEY,
      'X-API-Secret': process.env.PRIMEPIX_API_SECRET,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      amount,
      pixKey: 'user@example.com',
      pixKeyType: 'EMAIL'
    })
  });
  
  return withdrawResponse.json();
}
```

### Display balance in dashboard

```javascript React
import { useEffect, useState } from 'react';

function BalanceDisplay() {
  const [balance, setBalance] = useState(null);
  
  useEffect(() => {
    async function fetchBalance() {
      const response = await fetch('https://api.primepix.me/v1/balance', {
        headers: {
          'X-API-Key': process.env.PRIMEPIX_API_KEY,
          'X-API-Secret': process.env.PRIMEPIX_API_SECRET
        }
      });
      
      const { data } = await response.json();
      setBalance(data);
    }
    
    fetchBalance();
    
    // Refresh every 30 seconds
    const interval = setInterval(fetchBalance, 30000);
    return () => clearInterval(interval);
  }, []);
  
  if (!balance) return <div>Loading...</div>;
  
  return (
    <div>
      <h2>Account Balance</h2>
      <p>Available: R$ {balance.available.toFixed(2)}</p>
      <p>Pending: R$ {balance.pending.toFixed(2)}</p>
      <p>Total: R$ {balance.total.toFixed(2)}</p>
    </div>
  );
}
```
