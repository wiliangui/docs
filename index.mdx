## PrimePix API v1

A **PrimePix API** permite criar cobranças PIX, acompanhar pagamentos em tempo real e gerenciar o saldo da sua conta de forma simples e segura.

Exemplo de resposta do endpoint raiz da API:

```json
{
  "status": "online",
  "message": "PrimePix API v1 está funcionando",
  "version": "1.0.0",
  "baseUrl": "https://api.primepix.me/v1",
  "endpoints": {
    "pix_create": "/v1/pix/create",
    "pix_status": "/v1/pix/status/:id",
    "balance": "/v1/balance",
    "transactions": "/v1/transactions",
    "withdraw": "/v1/withdraw"
  }
}
```
