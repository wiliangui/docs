# Welcome to PrimePix

PrimePix is a complete PIX payment gateway that allows you to receive payments, manage your balance, and process withdrawals through a simple REST API.

## Key features

- **Instant PIX payments**: Generate QR codes and receive payments in real-time
- **Transaction management**: Track all your transactions with detailed filtering
- **Balance control**: Check your available balance and request withdrawals
- **Secure authentication**: API key and secret-based authentication
- **Webhooks**: Receive real-time notifications about payment status changes
