# Quickstart

## 1) Pegue suas credenciais

No painel: **Configurações → API**  
Você terá:

- `X-API-Key` (obrigatório)
- `X-API-Secret` (opcional, recomendado)

## 2) Criar uma cobrança PIX

```bash
curl -X POST "https://api.primepix.me/v1/pix/create" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: SUA_API_KEY" \
  -H "X-API-Secret: SEU_API_SECRET" \
  -d '{
    "amount": 150,
    "externalId": "pedido-12345",
    "description": "Compra na loja online",
    "expiresIn": 60,
    "customer": {
      "name": "João da Silva",
      "document": "123.456.789-00",
      "email": "joao@email.com"
    }
  }'
```

Você receberá `qrCode` (copia e cola) e `qrCodeBase64` (imagem).

## 3) Consultar o status

```bash
curl -X GET "https://api.primepix.me/v1/pix/status/pedido-12345" \
  -H "X-API-Key: SUA_API_KEY" \
  -H "X-API-Secret: SEU_API_SECRET"
```

Quando `status` virar `completed`, o pagamento foi confirmado.

## 4) (Recomendado) Use webhooks

Em produção, prefira webhooks ao invés de ficar consultando status.
Veja: [/webhooks/overview](/webhooks/overview)
