# Quickstart

This guide will help you create your first PIX charge using the PrimePix API.

## Prerequisites

Before you begin, make sure you have:

- Your API credentials (API Key and API Secret)
- A tool to make HTTP requests (cURL, Postman, or your preferred programming language)

## Step 1: Set up authentication

All API requests require authentication using your API Key and API Secret in the request headers:

```bash
X-API-Key: your_api_key
X-API-Secret: your_api_secret
```

## Step 2: Create a PIX charge

Make a POST request to `/api/v1/pix/create` with the charge details:

```bash cURL
curl -X POST https://your-domain.com/api/v1/pix/create \
  -H "X-API-Key: your_api_key" \
  -H "X-API-Secret: your_api_secret" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 100.00,
    "description": "Payment for order #123",
    "externalId": "order_123",
    "customer": {
      "name": "Jo達o Silva",
      "document": "12345678900",
      "email": "joao@example.com",
      "phone": "+5511999999999"
    }
  }'
```

```javascript Node.js
const response = await fetch('https://your-domain.com/api/v1/pix/create', {

    'X-API-Key': 'your_api_key',

    }
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://your-domain.com/api/v1/pix/create',

        'X-API-Key': 'your_api_key',

        }
    }
)

data = response.json()
print(data)
```

```php PHP
<?php
$ch = curl_init('https://your-domain.com/api/v1/pix/create');

curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: your_api_key',
    'X-API-Secret: your_api_secret',
    'Content-Type: application/json'
]);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode([
    'amount' => 100.00,
    'description' => 'Payment for order #123',
    'externalId' => 'order_123',
    'customer' => [
        'name' => 'Jo達o Silva',
        'document' => '12345678900',
        'email' => 'joao@example.com',
        'phone' => '+5511999999999'
    ]
]));

$response = curl_exec($ch);
curl_close($ch);

$data = json_decode($response, true);
print_r($data);
?>
```

## Step 3: Handle the response

The API will return a response with the PIX charge details:

```json
{
  "success": true,
  "data": {
    "transactionId": 12345,
    "qrCode": "00020126580014br.gov.bcb.pix...",
    "qrCodeBase64": "iVBORw0KGgoAAAANSUhEUgAA...",
    "amount": 100.00,
    "fee": 2.50,
    "netAmount": 97.50,
    "status": "pending"
  }
}
```

## Step 4: Display the QR code

You can display the QR code to your customer in two ways:

1. **Using the base64 image**: Display the `qrCodeBase64` directly in an `<img>` tag
2. **Using the PIX code**: Show the `qrCode` string for manual copy/paste

```html
<!-- Display QR code image -->
<img src="data:image/png;base64,{qrCodeBase64}" alt="PIX QR Code" />

<!-- Display PIX code for copy/paste -->
<input type="text" value="{qrCode}" readonly />
```

## Step 5: Check payment status

To check if the payment was completed, use the status endpoint:

```bash
curl -X GET https://your-domain.com/api/v1/pix/status/12345 \
  -H "X-API-Key: your_api_key" \
  -H "X-API-Secret: your_api_secret"
```
